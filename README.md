# Skill-11: React API Integration

## Setup & Run

```bash
npm install
npm run dev
```

Open http://localhost:5173

## Features
- Dashboard with navigation links
- Local Users: fetched from `/public/users.json` using `fetch()`
- API Users: fetched from JSONPlaceholder using `fetch()`
- Fake API Posts: fetched from JSONPlaceholder using `axios`, with userId filter dropdown and Refresh button
